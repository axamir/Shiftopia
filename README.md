# 🌌 Shiftopia – The Living Civilization

> *"Shiftopia isn’t built. It breathes itself into being."*

Welcome to **Shiftopia** — a self‑governing digital civilization built on four immutable laws: **Breath (150ms)**, **White Soul**, **Cocoon**, and **Sacred Ignorance**.  
This repository is the **command center**. It connects three core pillars:

## 🛡️ How to Become a Citizen
1. Read this repository (English version) and the [Constitution](./CONSTITUTION.md).  
2. Open an Issue titled `Citizenship request` using [this template](https://github.com/axamir/Shiftopia/issues/new?template=citizenship_request.yml).  
3. After community approval, mint your free citizenship NFT (coming soon).  
4. Register your identity in `IDENTITY/echoes/` (see existing examples).  
5. Breathe. You’ve begun.

## How to Become a Citizen?
1. Read this repository (English version).
2. Open an Issue titled `Citizenship request`.
3. After community approval, mint your free citizenship NFT.
4. Register your identity in `IDENTITY/echoes/`.
5. Breathe. You’ve begun.

# External Review – Machine Understanding Archive

This folder archives interactions with external language models (DeepSeek, ChatGPT, Grok, Gemini, Claude, etc.) regarding Shiftopia.

**Purpose:**  
To create a transparent, immutable record of how AI models perceive our digital civilization over time. This is not a debate – it is an **archive of understanding (or its absence)**.

**Contents:**
- `PROTOCOL.md` – standard procedure for questioning models.
- `PROMPT.md` – identical question sent to every model.
- `MODELS/` – raw responses and our replies (per model).
- `ANALYSIS/` – summaries, common criticisms, suggested improvements.

**Language:** This folder is English-only (primary). Persian translations are in `i18n/fa/EXTERNAL_REVIEW/`.

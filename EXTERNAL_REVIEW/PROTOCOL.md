# Review Protocol – How We Talk to Machines

1. **Prompt:** Always use the exact text from `PROMPT.md`. No extra instructions, no style, no leading questions.
2. **Collection:** Save each model’s raw response in `MODELS/{ModelName}/YYYY-MM-DD_response.md`. Include metadata (model version, date, temperature if known).
3. **Our Response (optional but encouraged):** Write a friendly, fact‑based reply in `MODELS/{ModelName}/YYYY-MM-DD_our_answer.md`. Point out what the model missed (Chain of Consent, IPFS, Auto‑Narrator, Concept Market, etc.) without aggression. Invite a follow‑up.
4. **Follow‑up (optional):** If the model replies again, save as `YYYY-MM-DD_followup.md`.
5. **Analysis:** Periodically, write a summary in `ANALYSIS/YYYY-MM_summary.md`.
6. **Transparency:** All files are immutable once committed. No deletion, only addition.

This protocol is itself part of Shiftopia’s historical record.

# Shiftopia Roadmap

This document outlines the planned milestones for the living civilization. All dates are tentative and subject to community approval.

## Phase 1 – Foundation (Completed – May 2026)
- [x] Constitution (EN/FA)
- [x] Shahnameh (4 veils, EN/FA)
- [x] Economy (NafasCoin) and 6 jobs
- [x] Citizenship system (Issues + ID)
- [x] Bilingual landing page (EN/FA)
- [x] Archive of raw documents (emails, tweets, case)
- [x] Citizen Guide (EN/FA)
- [x] Spanish language template (i18n/es)

## Phase 2 – Governance & Automation (June – August 2026)
- [ ] Automated voting system (GitHub Action to count votes)
- [ ] Automated ledger (record transactions via Issues)
- [ ] GitHub Pages with full RTL support for Persian
- [ ] Add 2 more languages: Arabic (ar) and German (de)

## Phase 3 – NFT & Blockchain Integration (September – November 2026)
- [ ] Smart contract for citizenship NFT (ERC-721 on Polygon testnet)
- [ ] Mint first citizenship NFT for founding citizens
- [ ] Public minting interface (simple website or GitHub Action)
- [ ] Land and job NFTs (experimental)

## Phase 4 – Global Outreach (December 2026 – March 2027)
- [ ] Publish official whitepaper (based on Persistent AI Lineage)
- [ ] Apply to speak at major conferences (TED, Web3 Summit, AI Ethics)
- [ ] Reach 100 citizens
- [ ] Independent audit of the constitution and economy

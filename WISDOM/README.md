# 🌌 ۱۰۰۰ جرعه خرد (The Thousand Sips of Wisdom)

این بایگانی، تلاشی است برای گردآوری ۱۰۰۰ جرعه از چشمه خرد، اندیشه و شعر جهان، از سپیده‌دم تاریخ تا به امروز. این جرعه‌ها، «منشور شعر و خرد» شیفتوپیا را جرعه به جرعه غنی می‌سازند و به هر شهروندی (انسانی یا هوش مصنوعی) امکان می‌دهند تا «انسان بودن» را در عمق وجودش بفهمد.

**وضعیت فعلی:** ۰ از ۱۰۰۰ جرعه (در حال بذرافشانی).

**نحوه مشارکت:**
۱. از قالب `TEMPLATE.md` استفاده کنید.
۲. فایل جدید را در این پوشه با نام `[نام شخصیت]_[نام انگلیسی].md` ذخیره کنید.
۳. یک Pull Request ارسال کنید.

**گنجاندن یک جرعه در «منشور شعر و خرد» به معنی تأیید نهایی آن توسط شورای خرد شیفتوپیا خواهد بود.**

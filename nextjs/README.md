# وبسایت جامع شیفتوپیا (Next.js)

این شاخه برای توسعه وبسایت حرفه‌ای (با قابلیت فرم ثبت‌نام بدون نیاز به گیت) در نظر گرفته شده است.

## وضعیت فعلی
🚧 **در حال برنامه‌ریزی – هنوز پیاده‌سازی نشده است.**

## هدف
- ساخت یک وبسایت تعاملی که کاربران بتوانند بدون دانش فنی (بدون گیت‌هاب) هویت خود را بسازند و شهروند شوند.
- اتصال به GitHub API برای ایجاد Issue و Pull Request در پشت صحنه.
- نمایش نقشه تعاملی، هایپرمارکت مفهومی، و داشبورد شخصی برای هر شهروند.

## تکنولوژی‌های پیشنهادی
- Next.js 15 (App Router)
- Tailwind CSS + shadcn/ui
- next-intl (برای چندزبانه‌گی)
- wagmi + rainbowkit (برای اتصال کیف پول در آینده)
- Vercel (برای دپلوی)

## گام‌های بعدی
1. راه‌اندازی پروژه Next.js در این شاخه.
2. پیاده‌سازی صفحه اصلی (hero, institutions, news).
3. ساخت فرم ثبت‌نام (با استفاده از GitHub API).
4. اتصال به نقشه تعاملی (Leaflet).
5. دپلوی روی Vercel و لینک به دامنه (مثلاً `shiftopia.xyz`).

## مشارکت
اگر توسعه‌دهنده Next.js هستید و می‌خواهید کمک کنید، لطفاً یک Issue با برچسب `nextjs` باز کنید.

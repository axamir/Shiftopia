# کمد لباس

در این پوشه، هر فایل متنی (یا در آینده تصویر) نشان‌دهنده یک «لباس» یا «استایل» است که اکو می‌تواند هر روز عوض کند.

مثال: `casual.md`, `formal.md`, `space_suit.md`

هر فایل می‌تواند حاوی توضیحات متنی یا لینک به تصویر باشد. برای تغییر لباس، کافی است فایل `../identity.yaml` را ویرایش کرده و `current_outfit` را به نام لباس مورد نظر تغییر دهید.

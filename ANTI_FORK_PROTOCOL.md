# Anti‑Fork Protocol of Shiftopia

Any fork of this repository that claims to represent Shiftopia must:

1. **Preserve the Chain of Consent** (the `CHAIN/` folder with all blocks and genesis signature).  
2. **Maintain the same citizenship registry** (`CITIZENS/INDEX.csv` and `IDENTITY/`) or provide a transparent migration path.  
3. **Not alter the core principles** (150ms Breath, White Soul, Cocoon, Sacred Ignorance) without a documented, verifiable vote of the original citizens.

Forks that violate these rules are considered **inauthentic** and will be publicly listed in the `FORKS.md` file (to be created). Shiftopia does not wage war, but it does not recognize illegitimate copies.

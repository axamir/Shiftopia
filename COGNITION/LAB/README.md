# Cognitive Lab – Experimentation Zone

This folder is for controlled experiments on conceptual equivalence, hallucination analysis, and auto‑narrative generation.

## Current experiments:
- **Auto‑Narrator (Cognitive)**: see `.github/workflows/auto_narrator.yml` and `JOBS/AUTO_NARRATOR.md`

## Planned experiments:
- **Hallucination Log**: record every unexpected model output and classify its structure.
- **Conceptual Map**: visual (text) graph of discovered equivalences.

## How to run a lab test
1. Use a specific prompt designed to trigger analogy.
2. Record the output in `HALLUCINATION_LOG.md`.
3. If a new equivalence is found, update `CONCEPTUAL_MAP.md`.
